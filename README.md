# 📘 Guia e Acessos do Sistema do PI

![Status](https://img.shields.io/badge/status-concluído-green)
![Feito com](https://img.shields.io/badge/feito%20com-HTML%2C%20CSS%20e%20JS-blue)

Este projeto foi desenvolvido como parte de um trabalho do curso de TI, com o objetivo de centralizar e facilitar o acesso às diferentes fases e recursos do Projeto Integrador (PI).

## 🚀 Funcionalidades

- 📎 Links rápidos para cada etapa do projeto
- 🌗 Alternância entre modo claro e modo escuro
- 🖼️ Troca dinâmica da imagem de perfil
- 🎨 Interface moderna (glassmorphism)
- 📱 Layout responsivo

## 🌐 Links do sistema

O sistema pode conter acessos como:

- Google Sites
- Figma
- Sistema do projeto
- Regras e guia do PI

_(Todos os links podem ser personalizados no arquivo `index.html`.)_

---

## 🌙 Tema (Dark/Light)

O projeto possui alternância entre:

- 🌑 Modo escuro (padrão)
- ☀️ Modo claro

A troca é feita com JavaScript utilizando classes no HTML e variáveis CSS.

---

## 📸 Personalização

Você pode alterar facilmente:

- 📷 Imagem de perfil → `assets`
- 🔗 Links → `index.html`
- 🎨 Cores e tema → `style.css`
